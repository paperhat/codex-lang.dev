# Codex Glossary (V-Z)

This file contains glossary entries for terms starting with V-Z.

See `../index.md` for format notes and other ranges.

[A–C](../a-c/index.md) | [D–F](../d-f/index.md) | [G–I](../g-i/index.md) | [J–L](../j-l/index.md) | [M–O](../m-o/index.md) | [P–R](../p-r/index.md) | [S–U](../s-u/index.md) | **V–Z**

## Terms

### ValidatorDefinition

A schema concept that defines a named validator used by ValueIsValid.

---

### ValidatorDefinitions

The container Concept in simplified schemas that holds ValidatorDefinition entries.

---

### Validity

Whether a well-formed document satisfies the semantic rules of its governing schema.

---

### Value

A literal datum spelled using the value literals defined by the specification and not evaluated as an expression.

---

### Value type

A schema-declared expected kind of Value, expressed with built-in tokens and optional parameters.

---

### Value type token

A built-in token (for example, `$Text`, `$Integer`, `$List`) that names a Value type.

---

### ValueInNumericRange

A value constraint requiring a numeric value to fall within an inclusive range.

---

### ValueIsNonEmpty

A value constraint requiring a text-like value to be present and non-empty.

---

### ValueIsOneOf

A value constraint requiring a value to be one of a listed set.

---

### ValueIsValid

A value constraint requiring a named validator to accept the value.

---

### ValueLength

A value constraint limiting the length of a text value.

---

### ValueMatchesPattern

A value constraint requiring a value to match a regular expression.

---

### ValueTypeDefinition

A schema concept that defines a custom value type.

---

### ValueTypeDefinitions

The container Concept in simplified schemas that holds ValueTypeDefinition entries.

---

### Version scheme

The schema trait that defines how version identifiers are compared within a lineage.

---

### Well-formedness

Whether a document is syntactically and structurally correct under the Codex surface form rules.

---

### Workshop

Paperhat Workshop (Workshop), a Paperhat product and consuming system that can interpret Codex and Gloss.

---
