## Terms

### key trait

The language-level trait that assigns a document-scoped lookup token to a Concept.

---

### List Value

An ordered collection value spelled with square brackets (`[...]`).

---

### Lookup token resolution

The process of resolving a lookup token to an `id` using the resolution table.

---

### Lookup Token Value

A `~name` token that refers to a Concept by its `key` and resolves within the document.
